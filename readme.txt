== Description ==

Browsable WP Tags is a simple wordpress plugin that will extract all your post tags and create a browsable list.

<strong>Here are a few blogs that are using our plugin:</strong>

<br>
<a href="http://www.rawmood.com/popular-tags" target="_blank">Raw Food Diet - Popular Tags</a>
<br>

== Installation ==

<strong>Install instructions</strong>
<ol>
	<li> Download the plugin from here</li>
	<li> Create the tags list.<ul>
        <li>Create a new page.</li>
	<li> Add your desired page title (ie. Tags list).</li>
	<li> In the content section add the following text: <strong>&lt;!–WP_Tags_List–&gt;</strong></li>
       </ul></li>
	<li> Save and publish the page</li>
	<li> Enjoy! :)</li>
</ol>
